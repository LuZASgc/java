内部类
======
内部类一定是在某个类花括号内部定义的类。
##编译结果
如果类A有一内部类B(A相对的可以称为外部类)。那么编译后，内部类的名称为`A$B.class`。
##非静态内部类
如果内部类B是非静态的。那么在外部类A以外实例化的时候，首先需要保证内部类在要实例化的类中可见（包可见，public，等）。
然后：
```java
A a = new A();
A.B b = a.new B();
```
注意一定要有一个实例化的外部类A的对象才行。
###静态内部类
如果内部类是静态的，且可见。那么：
```java
A.B b = new A.B();
```
简化了很多
###导入内部类
内部类也可以导入。
如果类A在包test中。那么我们在要实例化B类的程序(C类)中，可以这样：
```java
import test.A.B;

public class C {
    public static void main(String[]args) {
        //前提是B是静态的。
        B b = new B();
    }
}
```
可以发现，与一般类导入无异。
